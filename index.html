Projects

  • Restaurant Reviewsâ€™ Aggregator using Python

    Implemented a web crawler in Python that fetches restaurant reviews from the top review sites namely Yelp, Foursquare and Trip Advisor and classfied them based on the sentiment. Three types of applications, including a web application, an android application and a chrome plugin, were built to display the aggregated reviews using an option to search the restaurants. We also analysed the effectiveness of the three solutions with the help of user surveys and feedbacks.

  • Topic Classification of News Articles using Hadoop and Apache Spark

    In this project, we built a real time application that analyzes real time news feeds from various sources and appropriately classifies them into different categories. The news feeds are obtained by monitoring the RSS feeds of the news websites. The classification model is trained with huge data sets obtained from Wikipedia. Python libraries like NLTK and Scikit learn were used to train the model and then classify the news articles.

  • Android Mobile Application in Java

    Developed an Android mobile application that would facilitate our college students to create study groups and plan group meetings by reserving a room at the NCSU Hunt Library. The app provides the option to register, login, form study groups and plan group meetings. The user will also be redirected to the room reservation page of the library to book a room. Further the members of the study group will be notified the date, time and the location of the meetings.

  • Disk-Based Merge Sort using C++

    Implemented a disk-based merge sort algorithm to sort large data inputs that exceed the size of the main memory. The algorithm sorts the input in chunks of memory size and writes them to individual files. It then reads a part of the data from each file and sorts them using tournament sort. The results are stored in a temporary buffer and written to the final output file once the buffer is full. In this way, large data sets can be sorted with read and write overheads.
